# Системный монитор CPU

Десктопное приложение с GUI для мониторинга загрузки центрального процессора в реальном времени. Отображает текущую загрузку ЦП (%), тактовую частоту, количество работающих процессов и интерактивный скользящий график активности за последние 60 секунд.

Собрано на PyQt5 + pyqtgraph с бэкендом на psutil. Может быть упаковано в standalone исполняемый файл через PyInstaller.

## 🛠 Технологический стек

| Категория | Технология |
|-----------|-----------|
| **Язык** | Python 3.12+ |
| **GUI** | PyQt5 |
| **Графики** | pyqtgraph + NumPy |
| **Метрики системы** | psutil |
| **Сборка** | PyInstaller |

## 🚀 Ключевой функционал

- **Мониторинг CPU в реальном времени** — загрузка процессора обновляется каждую секунду.
- **Интерактивный график** — скользящее окно на 60 последних замеров, автошкала 0–100%.
- **Информационная панель:**
  - загрузка ЦП (%);
  - тактовая частота CPU (МГц);
  - количество работающих процессов.
- **Кнопка «Старт / Стоп»** — приостановка и возобновление сбора метрик.
- **Адаптация под системную тему** — автоматическое переключение тёмной/светлой палитры.
- **Standalone бинарник** — сборка в один .exe / ELF без потребности в интерпретаторе Python.

### Структура модулей

| Модуль | Ответственность |
|--------|----------------|
| `monitor.py` | Класс `SystemMonitor` — инкапсулирует все вызовы `psutil`, ведёт скользящую историю замеров. |
| `ui.py` | Класс `CPUMonitor(QWidget)` — вся GUI-логика: компоновка, график, таймер, тема. |
| `__main__.py` | Точка входа — создаёт `QApplication`, запускает окно. |

### Логика сбора данных

```
CPUMonitor (QWidget)
├── SystemMonitor                 # сборщик метрик
│   ├── get_cpu_percent()         # psutil.cpu_percent(interval=1)
│   ├── get_cpu_freq()            # psutil.cpu_freq().current
│   ├── get_process_count()       # len(psutil.process_iter())
│   └── poll() -> dict            # однократный опрос + история
├── QTimer (1000 ms)             # периодический вызов _update_stats()
├── pg.PlotWidget                # график, 60 точек, Y=0..100
└── Кнопка Старт/Стоп            # toggle running → timer.start/stop
```

## 💻 Локальное развертывание

```bash
# 1. Клонировать
git clone <repo-url>
cd linux-system-monitor

# 2. Создать виртуальное окружение
python3 -m venv venv

# 3. Активировать и установить зависимости
source venv/bin/activate          # Linux / macOS
# venv\Scripts\activate           # Windows

pip install -r requirements.txt

# 4. Запустить
python -m sys_monitor
```

### Сборка standalone-бинарника

```bash
source venv/bin/activate
pip install pyinstaller
pyinstaller sys_monitor/cpu-monitor.spec
```

Бинарник появится в `dist/`.

## 🔌 Использование

После запуска откроется окно приложения:

| Элемент | Описание |
|---------|----------|
| `Загрузка ЦП: 12.3%` | Процент загрузки CPU за последнюю секунду |
| `Текущая скорость: 3200.0 МГц` | Тактовая частота процессора |
| `Количество процессов: 425` | Количество процессов в системе |
| 📈 График | Скользящий 60-секундный график активности |
| 🔘 Стоп | Приостановить сбор метрик (возобновление — «Старт») |
