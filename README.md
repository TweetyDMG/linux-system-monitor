# Linux System Monitor — Мониторинг загрузки CPU в реальном времени

Десктопное приложение с GUI для мониторинга загрузки центрального процессора в реальном времени. Отображает текущую загрузку ЦП (%), тактовую частоту, количество работающих процессов и интерактивный скользящий график активности за последние 60 секунд. Собрано на PyQt5 + pyqtgraph с бэкендом на psutil.

---

## 🛠 Технологический стек

При проектировании архитектуры приложения упор делался на скорость обработки запросов и модульность.

*   **Язык разработки:** Python 3.12+
*   **GUI:** PyQt5
*   **Графики:** pyqtgraph + NumPy
*   **Метрики системы:** psutil
*   **Инструменты тестирования:** PyTest (планируется)
*   **DevOps:** PyInstaller (сборка standalone-бинарника)

---

## 🚀 Ключевой функционал

*   **Мониторинг CPU в реальном времени:** Загрузка процессора обновляется каждую секунду
*   **Интерактивный график:** Скользящее окно на 60 последних замеров, автошкала 0–100%
*   **Информационная панель:** Загрузка ЦП (%), тактовая частота (МГц), количество процессов
*   **Кнопка «Старт / Стоп»:** Приостановка и возобновление сбора метрик
*   **Адаптация под системную тему:** Автоматическое переключение тёмной/светлой палитры
*   **Standalone бинарник:** Сборка в один .exe / ELF через PyInstaller

## 📁 Архитектура и структура проекта

В проекте используется модульная архитектура с разделением на сбор метрик, GUI и точку входа.

```text
linux-system-monitor/
├── sys_monitor/                    # 📦 Пакет приложения
│   ├── __init__.py                 # Версия и метаданные пакета
│   ├── __main__.py                 # Точка входа: python -m sys_monitor
│   ├── monitor.py                  # Сбор метрик CPU через psutil
│   ├── ui.py                       # GUI-виджеты на PyQt5 + pyqtgraph
│   └── cpu-monitor.spec            # Spec-файл PyInstaller для сборки
├── venv/                           # 🐍 Виртуальное окружение Python
├── requirements.txt                # 📜 Зависимости проекта
├── LICENSE                         # ⚖️ MIT License
├── .gitignore                      # 🚫 Исключения Git
└── README.md                       # Текущая документация
```

### Логика сбора данных

```
CPUMonitor (QWidget)
├── SystemMonitor                   # сборщик метрик
│   ├── get_cpu_percent()           # psutil.cpu_percent(interval=1)
│   ├── get_cpu_freq()              # psutil.cpu_freq().current
│   ├── get_process_count()         # len(psutil.process_iter())
│   └── poll() -> dict              # однократный опрос + история
├── QTimer (1000 ms)               # периодический вызов _update_stats()
├── pg.PlotWidget                  # график, 60 точек, Y=0..100
└── Кнопка Старт/Стоп             # toggle running → timer.start/stop
```

## 💻 Локальное развертывание

Для запуска проекта в изолированном окружении вам понадобятся **Python 3.12+** и **Git**.

### 1. Клонирование репозитория

```bash
git clone https://github.com/<ваш-username>/linux-system-monitor.git
cd linux-system-monitor
```

### 2. Настройка переменных окружения

Переменные окружения не требуются — все настройки задаются через GUI.

### 3. Установка и запуск

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python -m sys_monitor
```

### 4. Сборка standalone-бинарника

```bash
source venv/bin/activate
pip install pyinstaller
pyinstaller sys_monitor/cpu-monitor.spec
```

Бинарник появится в `dist/`.

## 🔌 Использование приложения

После запуска откроется окно приложения:

| Элемент | Описание |
